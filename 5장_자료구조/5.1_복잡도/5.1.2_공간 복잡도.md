# 5.1.2 공간 복잡도 (Space Complexity)
## 1. 공간 복잡도
- 프로그램을 실행시켰을 때 **필요한 자원 공간의 양**

- 총 필요 저장 공간 : 고정 공간과 가변 공간으로 구분
- **고정 공간** (알고리즘과 무관한 공간)
  - 코드 저장 공간, 단순 변수 및 상수

- **가변 공간** (알고리즘 실행시 필요한 공간)

  - 실행 중 동적으로 필요한 공간
<br><br><br>

## 2. 공간 복잡도 계산 방법
  ```python
  S(P) = c + Sp(n)
  ```
  - c : 고정 고간

  - Sp(n) : 가변 공간
  - 고정 공간은 상수이므로 **공간 복잡도는 가변 공간에 좌우된다.**
